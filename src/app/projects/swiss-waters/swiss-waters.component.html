Schweizer Gewässer Daten - Immer informiert über die aktuellen Messungen:

Die Applikation "Schweizer Gewässer Daten" ermöglicht es Anglern und anderen Interessierten, die aktuellen Messdaten verschiedener Gewässer in der Schweiz abzurufen. Hier sind die Hauptfunktionen der Anwendung:

Startseite:

  • Die Startseite präsentiert alle verfügbaren Stationen mit den aktuellen Messdaten.
  • Jede Station wird durch eine Card repräsentiert, die die verfügbaren Messdaten anzeigt.
  • Eine Suchleiste erleichtert die Navigation durch alle Stationen.
  • Ein Dropdown-Menü nach Kantonen ermöglicht das Sortieren aller Gewässer nach Regionen.
  • Benutzer können ihre Lieblingsstationen markieren und nach diesen sortieren.

Detailansicht:

  • Durch Klicken auf "Details" können Benutzer eine detaillierte Ansicht jeder Station anzeigen.
  • Für jede verfügbare Messung werden der aktuelle Wert, der Mittelwert der letzten 24 Stunden, der maximale Wert der letzten 24 Stunden und der minimale Wert der letzten 24 Stunden angezeigt.
  • Wenn die Station Temperaturmesswerte hat, werden alle Messungen der letzten 7 Tage in einem Diagramm präsentiert.

Öffentliche API:

  • Die Applikation bietet eine öffentliche API, über die Benutzer Daten direkt als JSON abrufen können.
  • Durch die Bereitstellung eines API-Schlüssels können Entwickler die Daten in eigenen Anwendungen weiterverarbeiten.

Verwendete Technologien:

  • Frontend: Angular 13.3.0, Bootstrap 5
  • Backend: .NET 6 (RESTful API in C# mit GRPC)
  • Datenbank: Microsoft SQL

Umsetzung:

  • Das Frontend wurde mit Angular und Bootstrap für die Darstellung entwickelt.
  • Das Backend wurde als RESTful API mit .NET 6 in C# asynchron programmiert, wobei auch ein GRPC-Service für die Stationsdaten implementiert wurde.
  • Die Daten werden sicher in einer Microsoft SQL-Datenbank gespeichert.
