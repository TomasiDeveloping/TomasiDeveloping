Webseite Glasbläserei Egli

Die Firma Glasbläserei-Egli haben meine Grosseltern aufgebaut und wird von meiner Mutter als Einzelunternehmer weitergeführt.

Da man in der heutigen Zeit fast nicht um die Digitalisierung herumkommt, habe ich auch bei meiner Mutter einige Sachen digitalisiert und vereinfacht.

Um meine gelernten Kenntnisse weiter zu vertiefen und neues zu lernen, habe ich die Webseite Glasbläserei-Egli programmiert.

Beschreibung:

Die Website dient als Internetpräsenz für die Firma Glasbläserei-Egli, welche meiner Mutter gehört.

Auf der Seite wird die Geschichte der Firma beschrieben, sowie die Hauptprodukte vorgestellt.

Die Webseite hat zudem einen Admin Bereich, um Daten zu bearbeiten.

Verwendete Technologien:

  • Angular
  • HTML 5
  • CSS 3
  • Angular Material
  • ASP.NET
  • MSSQL

Umsetzung:

Die Webseite wurde mit Angular umgesetzt. Für das Design wurde zudem diverse Angular Material Komponente verwendet.

Es wurde ein Admin Bereich eingerichtet mit Login Funktion.

Für das Backend wurde eine RESTFull API mit .NET in C# programmiert.

Die Daten werden in einer Microsoft SQL-Datenbank gespeichert
