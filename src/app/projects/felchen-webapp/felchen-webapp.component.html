Felchen Webapp - Angeln leicht gemacht:

Die Felchen Webapp ist die Weiterentwicklung der Android Felchen App und wurde als Webanwendung komplett neu konzipiert. Diese Version bietet den Vorteil, dass sie als normale Webseite von jedem Gerät mit Internetzugang und Browser geöffnet werden kann, was die Plattformunabhängigkeit und schnellere Updates ermöglicht.

Funktionen:

  1. Sicherer Login:
    • Die App ist durch einen Login geschützt. Benutzer müssen sich entweder einloggen oder registrieren, um auf die Funktionen zuzugreifen.
    • Nach erfolgreichem Login erhält der Benutzer ein Json Web Token, das 1 Tag lang gültig ist.
  2. Fangübersicht:
    • Die Startseite zeigt dem Benutzer seine eingetragenen Fänge, standardmäßig nach Fangdatum sortiert.
    • Der Benutzer kann die Anzahl der Fänge pro Seite anpassen und nach Fangdatum sortieren.
    • Durch Klick auf "Detail zum Fang" gelangt der Benutzer zur Detailansicht des Fangs mit allen gespeicherten Daten.
  3. Öffentliche Fänge:
    • Der Tab "Öffentliche Fänge" zeigt alle Fänge an, die von Benutzern als öffentlich markiert wurden.
    • Durch Klick auf "Fang Detail" gelangt der Benutzer zur Detailansicht des Fanges.
  4. Detailansicht des Fangs:
    • Die Detailansicht zeigt alle gespeicherten Daten zu einem Fang.
  5. Fang bearbeiten:
    • Durch Klick auf "Fang bearbeiten" können alle Daten zu einem Fang bearbeitet werden.
    • Beim Bearbeiten werden keine aktuellen Wetterdaten geladen; alle Daten können manuell angepasst werden.
  6. Wetterinformationen:
    • Im Menü "Wetter" werden dem Benutzer die aktuellen Wetterdaten für seinen Standort angezeigt.
    • Wetterdaten fließen auch in einen neuen Fang ein.
    • Der Benutzer kann auch die Mondphasen der nächsten 7 Tage überprüfen.
  7. 7-Tage-Wetterprognose:
    • Die Funktion "7 Tage Prognose" ermöglicht dem Benutzer, die Wetterdaten für die nächsten Tage abzurufen.
    • Der Standort wird durch das Gerät ermittelt, und die Daten werden für die nächsten 7 Tage aufbereitet und angezeigt.

Verwendete Technologien:

  • Frontend: Angular, Bootstrap
  • Backend: .NET Core 5 (RESTful API in C#)
  • Datenbank: Microsoft SQL

Umsetzung:

Das Frontend wurde mit Angular und Bootstrap für die Darstellung entwickelt. Das Backend wurde als RESTful API mit .NET Core 5 in C# programmiert. Die gesamte REST-API wurde vollständig asynchron implementiert, um eine effiziente Kommunikation zu gewährleisten. Die Daten werden sicher in einer Microsoft SQL-Datenbank gespeichert.

Die Felchen Webapp steht zur Verfügung und kann gerne benutzt und getestet werden.
