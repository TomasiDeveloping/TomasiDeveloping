Meerschweinchen Gesundheits-Check

Mit dieser kleinen Webapplikation lassen sich Gesundheits-Checks für die Meerschweinchen speichern.

Jeder Check wird in der Datenbank gespeichert.

Zu jedem Meerschweinchen lassen sich Daten wie Name, Alter, Geschlecht und Rasse speichern.

Funktionen:

Die Applikation besteht aus einer Hauptseite, auf der alle Meerschweinchen aus der Datenbank angezeigt werden.

Es können weitere Meerschweinchen hinzugefügt werden

Mit dem Button bearbeiten kann jedes Meerschweinchen bearbeiten werden, um vorhandenen Daten anzupassen.

Mit Klick auf Neuer Check kann ein neuer Check für das ausgewählte Meerschweinchen hinzugefügt werden

Der Check besteht aus diversen Checkboxen, um auszuwählen was Geprüft wurde. Zu jedem Check kann wen gewünscht auch ein Kommentar hinzugefügt werden

Jeder Check hat ein Info Button, um eine kleine Beschreibung zum Check aufzurufen

Wurde der Info Button für ein Check geklickt erhält man eine kleine Beschreibung was bei diesem Test wichtig ist

Mit Klick auf Checks wird dem Benutzer die letzten 10 Checks für das ausgewählte Meerschweinchen angezeigt

In der Hauptanzeige sieht man das Datum und das Gewicht

Wurde für ein Check die Detail Ansicht geklickt sieht man den kompletten Check inklusive Kommentare sofern ein Kommentar für den Check hinterlegt wurde

Verwendete Technologien:

  • Angular 13.0.3
  • Bootstrap 5
  • .NET Core 6
  • MSSQL

Umsetzung:

Das Frontend wurde mit Angular programmiert und Bootstrap verwendet für die Darstellung

Für das Backend wurde eine RESTFull API mit .NET Core 6 in C# programmiert. Die REST wurde komplett asynchron programmiert

Die Daten werden in einer Microsoft SQL-Datenbank gespeichert.
