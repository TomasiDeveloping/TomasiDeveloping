Meerschweinchen Gesundheits-Check

Die Meerschweinchen Gesundheits-Check Webapplikation ermöglicht es Benutzern, Gesundheitsinformationen ihrer Meerschweinchen zu verfolgen und zu speichern. Die Anwendung wurde mit Angular für das Frontend und .NET Core in C# für das asynchrone Backend entwickelt. Bootstrap 5 wird verwendet, um eine benutzerfreundliche und ansprechende Benutzeroberfläche zu gewährleisten. Die Daten werden in einer Microsoft SQL-Datenbank gespeichert.

Funktionen:

  1. Hauptseite:
    • Alle Meerschweinchen in der Datenbank werden auf der Hauptseite angezeigt.
    • Für jedes Meerschweinchen können Name, Alter, Geschlecht und Rasse gespeichert werden.
    • Die Anzeige enthält Informationen wie das Datum des letzten Checks und das Gewicht.
  2. Meerschweinchen hinzufügen:
    • Benutzer können neue Meerschweinchen hinzufügen, indem sie relevante Informationen eingeben.
  3. Meerschweinchen bearbeiten:
    • Der "Bearbeiten"-Button ermöglicht es Benutzern, vorhandene Informationen zu aktualisieren.
  4. Neuer Check:
    • Mit einem Klick auf "Neuer Check" können Benutzer einen Gesundheits-Check für das ausgewählte Meerschweinchen hinzufügen.
    • Der Check umfasst verschiedene Checkboxen für zu prüfende Aspekte, und es kann ein Kommentar hinzugefügt werden.
  5. Info Button für Checks:
    • Jeder Check verfügt über einen "Info"-Button, der eine kurze Beschreibung des durchgeführten Tests liefert.
  6. Check-Historie:
    • Die Anwendung zeigt die letzten 10 Checks für das ausgewählte Meerschweinchen an.
    • Benutzer können das Datum und das Gewicht der Checks in der Hauptansicht sehen.
  7. Detailansicht für Checks:
    • Durch Klicken auf einen spezifischen Check können Benutzer alle Details einschließlich Kommentaren anzeigen, sofern vorhanden.

Verwendete Technologien:

  • Frontend: Angular, Bootstrap 5
  • Backend: .NET Core (C#), RESTful API (asynchron)
  • Datenbank: Microsoft SQL

Die Meerschweinchen Gesundheits-Check Webapplikation bietet eine benutzerfreundliche Oberfläche zur effektiven Verfolgung und Pflege der Gesundheit von Meerschweinchen.
