Fischer-Lizenz Manager - Online-Verwaltung für Fischereilizenzen:

Der Fischer-Lizenz Manager ist eine umfassende webbasierte Anwendung, die es Fischervereinen ermöglicht, ihre Fischereilizenzen online zu verwalten. Die Plattform bietet Funktionen zur Mitgliederverwaltung, Erfassung neuer Lizenzen und Online-Führung der Fangstatistik. Zusätzlich können verschiedene Excel-Dateien generiert werden, wie beispielsweise Mitgliederlisten, Lizenzübersichten und jährliche Fangstatistiken.

Beschreibung:

Die Applikation ist anpassbar und bietet einen personalisierten Bereich für Fischervereine. Die Startseite enthält Informationen über den eingeloggten Benutzer, die Vereinsadresse, Schonzeiten, Mindestmasse der Fische und Fischereivorschriften.

Funktionen:

  1. Lizenzen:
    • Mitglieder können ihre Lizenzen einsehen und das Ablaufdatum überprüfen.
    • Neue Lizenzen können vom Administrator ausgestellt und bearbeitet werden.
  2. Statistik:
    • Benutzer können ihre Fangstatistik online führen.
    • Administrator und Aufseher haben Zugriff auf die Fangstatistiken der Mitglieder.
  3. Fische und Schonzeiten:
    • Erfassung von Fischarten mit Mindestmaßen und Schonzeiten.
    • Automatische Anzeige auf der Startseite.
  4. Regeln und Vorschriften:
    • Definition eigener Regeln mit der Möglichkeit, Links zu PDFs oder Webseiten hinzuzufügen.
  5. Excel-Export:
    • Generierung von Excel-Dateien für Mitglieder, Lizenzen und jährliche Fangstatistiken.
  6. User-Verwaltung:
    • Administrator kann alle registrierten Benutzer überblicken und bearbeiten.
    • Deaktivierung von Benutzern und Vergabe von Administratorrechten.
  7. Aufseher-Übersicht:
    • Aufseher haben Zugriff auf die Lizenzen und Fangstatistiken der Mitglieder.

Verwendete Technologien:

  • Frontend: Angular, Bootstrap
  • Backend: .NET Core 5 (RESTful API in C#)
  • Datenbank: Microsoft SQL

Umsetzung:

Das Frontend wurde mit Angular und Bootstrap für die Darstellung entwickelt. Das Backend wurde als RESTful API mit .NET Core 5 in C# programmiert. Die gesamte REST-API wurde vollständig asynchron implementiert, um eine effiziente Kommunikation zu gewährleisten. Die Daten werden sicher in einer Microsoft SQL-Datenbank gespeichert.

Demo-Version:

Für Testzwecke steht eine Demo-Version zur Verfügung. Mit den Testusern Admin, Aufseher und User können die verschiedenen Rollen der Plattform getestet werden. Einige Features sind in der Demo-Version eingeschränkt.
