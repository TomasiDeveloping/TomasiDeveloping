Game Inventory

Da ich leidenschaftlicher Spielekonsole Sammler bin, besitze ich diverse Konsolen und Games.

Damit man nicht alle Spiele der Konsolen durchsuchen muss wen man ein Spiel sucht, oder nachschaut, ob man ein bestimmtes Spiel besitzt, hatte ich die Idee die GameInventory Applikation zu entwickeln, um seine Konsolen und Games zu verwalten

Es soll auf einfache Art ein Spiel gesucht werden oder Spiele für die gewünschte Konsole nach Genres auflisten.

Beschreibung:

Game Inventar ist eine Webbasierte Applikation, mit der man seine Gamesammlung managen kann.

In der Applikation kann man Publisher, Genres, Konsolen und Spielmodus erfassen. Ein Game kann dann beim Erfassen der gewünschte Publisher, die vorhandenen Konsolen die Spielmodus und die Genres hinzugefügt werden.

Im Hauptmenu können Spiele danach anhand der Konsolen, der Genres und den Spielmodus sortiert werden. Der Nutzer kann somit ein Game aus seiner Sammlung direkt suchen und sieht auf welchen Konsolen er das Spiel besitzt und in was für einem Medium (Disc, Digital usw.…)

Funktionen:

Im Hauptmenu der App werden alle Games in einer Liste angezeigt.

Für die Spiele wird ein Cover angezeigt, Name, Publisher und Erstveröffentlichung.

Die Games lassen sich nach den vorhandenen Konsolen sortieren.

Die Games können nach den erfassten Genres sortiert werden.

Alle Games lassen sich per Autocomplete suchen.

Mit doppelklick auf das gewünschte Game öffnet sich eine Detailanzeige mit allen Informationen zum Game.

Über die Detailanzeige lässt sich das gewünschte Game bearbeiten oder auch löschen.

Im Abschnitt Konsolen werden alle erfassten Konsolen des Benutzers angezeigt mit den erfassten Daten.

Jede erfasste Konsole lässt sich auch bearbeiten oder löschen.

Im Abschnitt Publisher sieht der Benutzer alle Publsiher die er erfasst hat.

Mit Klick auf einen Publisher erhält der Benutzer Informationen über den gewünschten Publisher und kann diesen bearbeiten oder löschen.

Im Abschnitt Genre sieht der Benutzer alle Genres, die er erfasst hat und kann diese bearbeiten oder löschen.

Im letzten Abschnitt Einstellungen hat der Benutzer noch eine Übersicht über alle erfassten Game Engines, Spielmodus und Mediums, diese lassen sich alle bearbeiten oder auch löschen.

Verwendete Technologien:

  • Angular
  • Angular Material
  • ASP.NET
  • MSSQL

Umsetzung:

Die Applikation wurde min Angular als Frontend umgesetzt. Es wurden zudem diverse Angular Material Komponenten eingesetzt.

Für das Backend wurde eine RESTFull API mit .NET in C# programmiert.

Über die Endpunkte der REST lassen sich alle Daten bearbeiten, Anzeigen sowie auch löschen.

Als Datenbank wurde eine Microsoft SQL-Datenbank gewählt.
