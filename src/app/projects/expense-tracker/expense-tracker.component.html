Kostenmanager - Behalten Sie Ihre Finanzen im Blick:

Die Kostenmanager-App ermöglicht es Ihnen, Ausgaben und Einnahmen bequem online zu erfassen und zu verwalten. Hier sind einige der Funktionen, die Ihnen die App bietet:

Funktionen:

  1. Übersichtliche Startseite:
    • Die Startseite zeigt immer den aktuellen Monat an.
    • Ein % Balken visualisiert die Ausgaben im Verhältnis zum Budget, wenn in den Einstellungen ein Budget gesetzt wurde.
    • Bei Nutzung von Einnahmen wird die Differenz zwischen Einnahmen und Ausgaben berechnet.
  2. Kategoriebasierte Ausgaben:
    • Alle Ein- und Ausgaben können in Kategorien unterteilt werden.
    • Die App zeigt Ausgaben pro Kategorie, berechnet den Prozentsatz der Kategorie zu den Gesamtausgaben und listet sie übersichtlich auf.
  3. Schnelle Erfassung von Ausgaben:
    • Mit einem Klick auf "Ausgabe erfassen" können Sie schnell eine neue Ausgabe eingeben.
    • Sie können die Kategorie auswählen oder eine neue Kategorie direkt erfassen.
  4. Ausgabenverwaltung:
    • Die Ausgabenseite zeigt alle Ausgaben des aktuellen Monats an.
    • Jede Ausgabe kann bearbeitet oder gelöscht werden.
    • Sie haben Zugriff auf Ausgaben bis zu 4 Jahre zurück oder 1 Jahr in die Zukunft.
  5. Kategorienverwaltung:
    • Auf der Kategorienseite können Sie Ihre erfassten Ein- und Ausgabenkategorien verwalten.
    • Neue Kategorien können hinzugefügt, bearbeitet oder gelöscht werden.
  6. Statistikübersicht:
    • Die Statistikseite zeigt Diagramme der Ausgaben für das aktuelle Jahr an.
    • Bei Nutzung von Einnahmen gibt es auch ein Diagramm für die Einnahmen und die Differenz zwischen Ein- und Ausgaben.
    • Sie können Jahres- und Monatsstatistiken als Excel herunterladen.
  7. Benutzereinstellungen:
    • In den Einstellungen kann der Benutzer seinen Account verwalten.
    • Es kann eingestellt werden, ob die Applikation nur mit Ausgaben geführt werden soll.
    • Das Monatsbudget kann hier angepasst werden.

Verwendete Technologien:

  • Frontend: Angular 13.1.1, Bootstrap 5
  • Backend: .NET 6 (RESTful API in C#)
  • Datenbank: Microsoft SQL

Umsetzung:

Das Frontend wurde mit Angular und Bootstrap für die Darstellung programmiert. Das Backend wurde als RESTful API mit .NET 6 in C# asynchron entwickelt. Die Daten werden sicher in einer Microsoft SQL-Datenbank gespeichert.
