Kostenmanager

Mit dieser App lassen sich Ausgaben wie auch Einnahmen bequem Online erfassen und Verwalten.

Ein und Ausgaben können in Kategorien unterteilt werden.

Alle Ein und Ausgaben können jederzeit bearbeitet oder gelöscht werden. Es kann zudem für jeden Monat oder das ganze Jahr eine Excel Statistik heruntergeladen werden.

Funktionen:

Auf der Startseite wird immer der aktuelle Monat angezeigt.

Wurde in den Einstellungen ein Budget gesetzt zeigt der % Balken immer die Ausgaben in % zum Budget an. Wurde in den Einstellungen gewählt das der Kostenmanager mit Einnahmen geführt wird, wird zudem die aktuelle Differenz zwischen Einnahmen und Ausgaben berechnet.

Im unteren Teil werden alle Ausgaben pro Kategorie sortiert und der Prozentsatz der Kategorie zu allen aktuellen Ausgaben berechnet und angezeigt.

Mit Klick auf Ausgabe Erfassen lässt sich schnell eine neue Ausgabe eintragen.

Bei der Kategorie stehen alle erfassten Kategorien zur Verfügung, oder es kann auch gleich eine neue Kategorie für die Ausgabe erfasst werden.

In den Ausgaben sieht man alle Ausgaben für den aktuellen Monat.

Alle Ausgaben für jede Kategorie werden aufgelistet mit dem Preis und der Gesamtsumme der Kategorie.

Jede Ausgabe kann hier bearbeitet oder auch gelöscht werden.

Es kann nach Wunsch auch bis 4 Jahre zurück oder 1 Jahr in die Zukunft gewählt werden, um die Ausgaben anzuschauen oder zu bearbeiten.

Auf der Kategorien Seite erhält der Benutzer eine Übersicht, über die Erfassten Kategorien für Ein und Ausgaben.

Jede Kategorie kann hier bearbeitet oder nach Wunsch auch wieder gelöscht werden.

Auf dieser Seite lassen sich auch neue Kategorien erfassen.

Auf der Statistik Seite erhält der Benutzer ein Diagramm mit den ganzen Ausgaben für das Aktuelle Jahr.

Wird die Applikation mit Einnahmen geführt, wird auch ein Diagramm für die Einnahmen angezeigt, sowie eine Berechnung der Differenz zwischen Ein und Ausgaben.

Der Benutzer kann nach Wunsch hier direkt eine Jahresstatistik als Excel herunterladen oder für einen gewünschten Monat eine Detaillierte Monatsstatistik.

Beispiel einer Detaillierter Monatsstatistik als Excel.

In den Einstellungen kann der Benutzer sein Account verwalten.

Es kann hier eingestellt werden, ob die Applikation nur mit Ausgaben geführt werden möchten, was eine Auswirkung auf die Anzeige, wie auch die Excel Statistiken hat. Auch das Monatsbudget kann hier jederzeit geändert werden.

Verwendete Technologien:

  • Angular 13.1.1
  • Bootstrap 5
  • .NET 6
  • MSSQL

Umsetzung:

Das Frontend wurde mit Angular programmiert und Bootstrap verwendet für die Darstellung

Für das Backend wurde eine RESTFull API mit .NET 6 in C# programmiert. Die REST wurde komplett asynchron programmiert

Die Daten werden in einer Microsoft SQL-Datenbank gespeichert.
