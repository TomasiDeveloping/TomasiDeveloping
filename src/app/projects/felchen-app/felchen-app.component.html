Felchen App - Die App für leidenschaftliche Felchen-Fischer:

Die Felchen App ist das Ergebnis meiner Ausbildung zum Applikationsentwickler, bei der ich die Herausforderung hatte, eine mobile App mit ReactNative zu entwickeln. Ursprünglich für den persönlichen Gebrauch konzipiert, wurde die App später komplett überarbeitet und mit Ionic neu implementiert. Das eigene Backend und eine Login-Funktion wurden hinzugefügt, um die App auf eine breitere Nutzerbasis auszurichten.

Beschreibung:

Die Felchen App bietet passionierten Felchen-Fischern die Möglichkeit, ihre Fänge systematisch zu speichern. Die App unterstützt die Komplexität der Felchen-Fischerei, bei der verschiedene Nymphen mit Unterschieden in Farbe, Hakengröße und anderen Merkmalen verwendet werden.

Funktionen:

  1. Statistik Screen:
    • Anzeigen aller gespeicherten Fänge nach Datum sortiert.
    • Zugriff auf den Detail Screen für jeden einzelnen Fang.
  2. Detail Screen:
    • Detaillierte Anzeige aller gespeicherten Daten für einen bestimmten Fang.
    • Möglichkeit, einen Fang zu löschen.
  3. Fang hinzufügen:
    • Erfassung neuer Fangdaten, darunter Wetterinformationen automatisch über eine API.
    • Manuelle Anpassung der Wetterdaten möglich.
  4. Wetterdaten:
    • Automatisches Laden von Wetterdaten per GPS oder manuelle Eingabe von PLZ oder Ort.
    • Dynamische Hintergrundänderungen basierend auf Wetter und Uhrzeit.
  5. Einstellungen:
    • Benutzer können persönliche Angaben wie Namen oder E-Mail-Adresse ändern.
    • Funktionen zum Ausloggen, Passwort ändern oder den Account löschen.

Vorgesehene Features:

  • Öffentliche Fänge anderer App-Benutzer.
  • Bearbeiten von gespeicherten Fängen.
  • Sortieren der Fänge.

Verwendete Technologien:

  • Frontend: Ionic
  • Backend: ASP.NET (RESTful API in C#)
  • Datenbank: Microsoft SQL

Umsetzung:

Die App wurde mit Ionic entwickelt und kommuniziert über das HTTP-Protokoll mit der RESTful API, die mit ASP.NET und C# erstellt wurde. Die Daten werden in einer Microsoft SQL-Datenbank gespeichert.

Mit der Felchen App steht eine benutzerfreundliche Plattform zur Verfügung, die nicht nur die Speicherung von Fängen ermöglicht, sondern auch durch die Integration von Wetterdaten und geplanten Features eine umfassende Erfahrung für Felchen-Fischer bietet.
